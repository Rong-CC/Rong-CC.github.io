# VSCODE文件头部注释插件KOROFILEHEADER配置

一、

​ vscode 插件安装 koro1FileHeader

二、配置

​ 如果不修改默认的配置头部注释效果如下,我想去掉Author和LastEditors,并调整顺序，配置完记得重启vscode

/***
 * Author       : ''
 * Date         : 2019-05-13 15:54:32
 * LastEditors  : ''
 * LastEditTime : 2020-02-14 22:04:52
 * FilePath     : /fileHead/index.html
 * Description  : 首页
 **/

如果需要自定义表头配置去设置中 搜索 fileheader

你只需要把fileheader.customMade和fileheader.cursorMode放进去就可以了。

  • fileheader.customMade 是头部文件注释的配置
  • fileheader.cursorMode 是函数注释的配置
  • prohibitAutoAdd设置md和json文件不自动添加头部注释
  • 如果你要了解更多的配置可以看koroFileHeader的官方文档
  • 要了解更多的编码规范可以参考javascript编码规范 (opens new window)
 "fileheader.customMade": { 
    "Author":"rongcheng",
    "Date": "Do not edit", // 文件创建时间(不变)
    "LastEditors": "", // 文件最后编辑者
    "LastEditTime": "Do not edit", // 文件最后编辑时间
    "@后台人员":"",
    "FilePath": "Do not edit",
    "Description":"",
  },

三、快捷键

win: ctrl+alt+i Mac: control+ command + i 添加 文件头 注释 win: ctrl+alt+t Mac: control+ command + t 添加 函数 注释

四、关闭自动文件注释

  "fileheader.customMade": { 
        "autoAdd": false,
   }
